# The Wild Oasis

The Wild Oasis is a modern cabin booking web application built as part of a learning project.  
It allows users to explore luxury cabins, check availability, make reservations, and manage their bookings through a personal guest area.

## Preview

A peaceful and elegant landing page introduces the platform with a strong visual focus on nature, comfort, and premium cabin experiences.

## Features

- Browse available cabins
- View cabin details
- Reserve cabins for selected dates
- Authentication for guests
- Personal guest area
- Manage existing reservations
- Update guest profile information
- Responsive and modern UI

## Tech Stack

This project is built with:

- **Next.js**
- **React**
- **JavaScript / TypeScript**
- **Tailwind CSS**
- **Supabase** for database and backend services
- **Authentication** for guest login
- **Server Actions** for handling mutations
- **date-fns** for date calculations

## Pages / Main Functionality

### Home Page

The landing page introduces users to the platform and encourages them to explore cabins.

### Cabins Page

Users can browse all available cabins and check their details.

### Cabin Details Page

Each cabin has its own page with more information, pricing, and reservation options.

### Guest Area

Authenticated users can:

- view their bookings
- update personal data
- manage reservations
